# 项目进存销管理系统 - 通讯施工队版

## 项目概述
为通讯施工团队（光缆布放、摄像头安装）打造的进存销管理系统，支持按项目核算成本与收益。

## 技术栈
- Next.js 14 (App Router, TypeScript)
- Tailwind CSS
- Cloudflare D1 (SQLite 数据库)
- Cloudflare R2 (文件存储)
- Cloudflare Pages (部署)
- Drizzle ORM (数据库 ORM)

## 角色权限模板

### 施工队模板
| 职位 | 权限 |
|------|------|
| 仓管 | 入库管理、库存查询 |
| 施工员 | 领料出库、查看库存 |

### 简单模板

|------|------|

## 物料分类
- 光缆类：光纤、光缆、网线等
- 设备类：摄像头、交换机、NVR、服务器
- 辅材类：接头盒、尾纤、电源线、支架、波纹管
- 工具类：熔接机、OTDR、脚手架

## 核心功能

### 1. 物料管理
- 物料台账：名称、规格、单位、采购价、结算价
- 库存预警：低于最小库存时提醒
- 分类管理：支持多级分类

### 2. 入库管理
- 供应商选择
- 多物料入库
- 单据照片上传
- 自动更新库存

### 3. 出库管理
- 关联项目
- 自动计算成本和产值
- 扣减库存
- 单据照片上传

### 4. 项目管理
- 项目信息维护
- 材料消耗明细
- 成本/产值/利润统计

### 5. 收益报表
- 日/周/月报表
- 按项目汇总
- 按物料统计
- 导出功能

### 6. 系统设置
- 职位权限配置
- 员工账号管理
- 数据备份

## 收益计算公式
- 产值 = 出库数量 × 结算价
- 成本 = 出库数量 × 采购价
- 利润 = 产值 - 成本

## 部署说明
1. Cloudflare Pages 创建项目
2. D1 数据库创建
3. R2 存储桶创建
4. 环境变量配置
5. 部署上线

## 开发命令
```bash
npm install          # 安装依赖
npm run dev          # 本地开发
npm run db:generate  # 生成 Drizzle 迁移
npm run db:migrate   # 执行数据库迁移
npm run db:seed      # 初始化种子数据
```
